Description of layout algorithm: Pipeline approach. First reducing crossings, then wiggle number, then wiggle distance, and whitespace. After locations are sorted greedily, crossings are optimized as in standard storylines but with the location restrictions. This is done in Sugiyama style, barycenter heuristic considering the hierarchy.
To minimize the wiggle count in the next step, the problem is heuristically reduced to the longest common subsequence problem.
Next, given the ordering constraints and constraints on some lines being straight, a quadratic program minimizes squared wiggle height and white space given two weights.
