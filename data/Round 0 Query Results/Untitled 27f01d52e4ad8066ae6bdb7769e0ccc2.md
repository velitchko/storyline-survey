Checked By: Alex, Henry
Description of layout algorithm: Minimization of block crossings, which are cross-
ings between two bundles of edges. They show poly for 0
block crossings, FPT in number of characters. Greedy al-
gorithm for meetings of size 2 which yields optimal solution
for 3 entities. Constant factor approximation for constant-size
meetings, where meetings cannot be repeated. Further, they also show NP-hardness of block crossing minimization. They also improve the FPT-algorithm for pair crossings parameterized by the number of characters.
