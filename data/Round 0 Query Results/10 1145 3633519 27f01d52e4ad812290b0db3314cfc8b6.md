Authors: Wang, Yunchao, Sun, Guodao, Zhu, Zihao, Li, Tong, Chen, Ling, Liang, Ronghua
Checked By: Alex, Henry
Description of layout algorithm: Multi-objective optimization ILP, where characters have discrete coordinates. Crossing minimization is main objective. Further, the objective can also contain sum of squared distance between pairs that are in an entity-entity relationship at that time. That is, events consist of multiple entitiy-entity relationships which are explicitly visualized in a matrix. In their evaluation, they only minimize crossings though.
Constraints / Assumptions: Discrete coordinates, Interaction grouping
Domain: Movement data - sport, Movies/Books
Evaluation: User study
Exclude?: No
Keep: Yes
Layout Method fully described in paper: Yes
Method Type: Exact - ILP
Optimization Objectives: Crossings - pair crossings, Proximity - Entity pair squared distance
Source: ACM
Special features: Entity pair relationship visualization
Temporal Model: Discrete, Persistent characters
Title: E2Storyline: Visualizing the Relationship with Triplet Entities and Event Discovery
