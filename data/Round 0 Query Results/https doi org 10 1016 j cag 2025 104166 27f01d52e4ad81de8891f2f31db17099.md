Description of layout algorithm: Focus on locations. First, actors, interactions, and locations, get a significance score (weight). The single location order is computed with MDS based on number of characters going between locations. Actors have a global order, and this order is used to sort actors in events. Then a barycenter-like post-processing is used. Y-coordinates do not need to be computed, as locations are rows and actors are equally spaced.
