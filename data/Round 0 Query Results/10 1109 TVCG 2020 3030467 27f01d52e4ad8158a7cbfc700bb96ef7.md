Description of layout algorithm: This is an extension to istoryline, where reinforcement learn-
ing is used to compute storylines which adhere to constraints
that are obtained from interactions.
The initial layout is computed the same as istoryline. Then the user modifies the layout, and the AI agent can propose different variants of new layouts, and the user can choose one of them.
Also, a new set of interactions ins introduced.

The AI agent tries to predict what a user would do to change the layout from the initial layout and the user-adapted layout.
