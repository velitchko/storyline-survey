Description of layout algorithm: Interactive tool to assist storyline drawing. So interactions impose constraints on the layout process. In general, the storyflow algorithm is used. 

When constraints on character order are imposed, a heuristic from [Forster, 2005 is used] which essentially is an adaptation of the barycenter heuristic. 

Constraints can also say that character coordinates must behave in a specific way, which is encoded in the alignment (wiggle count, wiggle height, whitespace reduction) step.

Further interactions also change the alignment step.
