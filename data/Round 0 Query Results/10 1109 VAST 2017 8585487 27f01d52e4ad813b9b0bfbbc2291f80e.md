Description of layout algorithm: The paper compares non-standard storyline (which does not fall within our definition) with standard storylines. The standard storyline algorithm: Crossings are mini-
mized by sweep + Median heuristic. Then a local search
swaps neighboring nodes. Wiggle count is minimized by
modelling as independent set (cannot use lcs because mul-
tiple in/out-edges) and heuristic for independent set is used.
Whitespace is reduced with linear programming.
For the case where entities are grouped into locations per
timesteps and locations have a fixed order: Single node per lo-
cation with edge-weighted directed graph. Greedy algorithm
is used.
